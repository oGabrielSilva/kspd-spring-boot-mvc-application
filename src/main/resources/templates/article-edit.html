Configure a imagem

Figcaption é o rótulo visível da imagem, exibido abaixo dela. É utilizado para descrever a imagem e/ou fornecer informações sobre sua fonte.

Metadados

Fornece informações adicionais sobre uma imagem quando o cursor do mouse está sobre ela, ajudando a contextualizar sua finalidade ou oferecer uma descrição mais detalhada
O atributo "alt" em HTML fornece um texto alternativo que é exibido quando a imagem não pode ser carregada
O rótulo é um texto que descreve a imagem. Ele é um atributo de acessibilidade. Nesse caso, não é obrigatório se a descrição no Figcaption for suficiente.
